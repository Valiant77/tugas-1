Tugas pertama kelas pra-magang bidang Web Programming di PT. Makerindo Prima Solusi, atas nama: Achmad Fachri Ramadhan

Bukutheca (CRUD)

##Requirement
- PHP 8.1+
- Composer
- MySQL

##Langkah Setup
1. Klon repositori ini
2. Jalankan 'composer install'
3. Copy '.env.example' ke '.env'
4. Jalankan 'php artisan key:generate'
5. Jalankan 'php artisan migrate'
6. Jalankan 'php artisan serve' atau jalankan menggunakan Apache dan buka http://bukutheca.test

##Fitur
- CRUD
- Relasi Many-to-Many
- UI Bootstrap (disambung dengan CDN)
